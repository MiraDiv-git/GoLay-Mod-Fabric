# GoLay
It's the [GoProne](https://www.curseforge.com/minecraft/mc-mods/goprone)-like mod, but it's made for Fabric. It's not a port or fork, so code is completely written by me.

## What is it for?
This mod allows you to crawl by pressing the button *(default: 'C')* or typing the /lay command. Yes, it makes you 1 block height, allowing you to crawl through narrow gaps.

## Usage
You can use binding *(button 'C')* to switch between crawling and not-crawling mode. <br>
Also if you're lazy to download mod to the client side, you can use commands:
- /lay - to lay
- /unlay - to stand up
